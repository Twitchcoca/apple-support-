Si vous avez oublié le mot de passe de votre compte Apple

Voici comment réinitialiser le mot de passe de votre compte Apple et retrouver l'accès à celui-ci.

Le moyen le plus simple de réinitialiser votre mot de passe est d'utiliser votre iPhone ou un autre appareil Apple de confiance, déjà connecté à votre compte Apple, afin qu'Apple puisse l'identifier. Vous devez également avoir configuré un code d'accès (ou un mot de passe sur Mac) sur cet appareil.

Réinitialiser votre mot de passe sur Internet lorsque vous ne disposez pas d'un appareil de confiance est une autre façon de récupérer l'accès à votre compte.

L'identifiant Apple s'appelle désormais Compte Apple. Vous pouvez toujours vous connecter avec la même adresse e-mail ou le même numéro de téléphone et le même mot de passe.

Réinitialisez le mot de passe sur votre appareil de confiance

Sur iPhone, iPad ou Apple Vision Pro :

  • Allez dans les Réglages, puis appuyez sur votre nom.
  • Appuyez sur Connexion et sécurité, puis sélectionnez Changer le mot de passe.
  • Suivez les instructions à l'écran.

Sur votre Mac :

  • Dans le menu Apple , choisissez Réglages système.
  • Cliquez sur votre nom, puis choisissez Connexion et sécurité.
  • Cliquez sur Changer le mot de passe.

Si vous ne possédez pas d'appareil de confiance

Vous pouvez emprunter un appareil Apple à un ami ou à un membre de votre famille et utiliser l'application Assistance Apple.

  • Ouvrez l’application Assistance Apple.
  • Faites défiler jusqu'à Outils d'assistance, puis appuyez sur Réinitialiser le mot de passe.
  • Appuyez sur « Aider quelqu'un d'autre ».
